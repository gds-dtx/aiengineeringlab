# Evaluating AI agents

Anthropic's engineering blog post [Demystifying evals for AI agents](https://www.anthropic.com/engineering/demystifying-evals-for-ai-agents) explains how to design and run evaluations for agentic systems.

It covers the difference between task-completion, trajectory, and component evals, how to use a language model as a judge, and common pitfalls such as overfitting to the eval and single-metric thinking.

Read it if you are building or assessing agents with Claude and want a practical framework for measuring whether they actually work.
